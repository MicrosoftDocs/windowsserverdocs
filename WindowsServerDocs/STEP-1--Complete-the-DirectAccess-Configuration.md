# STEP 1: Complete the DirectAccess Configuration
The first step is to complete all the steps in the [Test Lab Guide: Demonstrate DirectAccess Single Server Setup with Mixed IPv4 and IPv6](http://go.microsoft.com/fwlink/p/?LinkId=237004). If you have already completed the steps in this test lab guides and saved a snapshot or disk image of the test lab, you can restore the snapshot or image and begin with the next step.
