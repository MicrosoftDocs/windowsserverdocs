# bitsadmin peers and list

Applies to: Windows Server (All supported versions)

The `bitsadmin` command when combined with the **peers** and **list** switches lists all peers.

## Syntax

```CLI
bitsadmin /peers /list
```

## Examples

To list all peers:

```CLI
bitsadmin /peers /list
```
