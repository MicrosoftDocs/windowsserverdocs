# nslookup /exit

Exits the nslookup command-line tool.

## Syntax

```
nslookup /exit
```

### Parameters

| Parameter | Description |
| --------- | ----------- |
| /? | Displays help at the command prompt. |
| /help | Displays help at the command prompt. |
