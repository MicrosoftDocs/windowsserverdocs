# ftp bye



Ends the ftp session on the remote computer, and then exits.

> [!NOTE]
> This command is the same as the [ftp quit command](ftp-quit.md).

## Syntax

```
bye
```

### Examples

To end the ftp session with the remote computer and exit, type:

```
bye
```
