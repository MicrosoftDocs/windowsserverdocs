# bitsadmin makecustomheaderswriteonly

Make a job's Custom HTTP Headers write-only (cannot be undone).

## Syntax

```
bitsadmin /MakeCustomHeadersWriteOnly <Job>
```

## Parameters

|Parameter|Description|
|---------|-----------|
|Job|The job's display name or GUID|
