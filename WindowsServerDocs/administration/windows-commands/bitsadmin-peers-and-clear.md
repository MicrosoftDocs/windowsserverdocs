# bitsadmin peers and clear



Clears the peer list.

## Syntax

```
bitsadmin /Peers /Clear 
```

## <a name="BKMK_examples"></a>Examples

The following example clears the peer list.
```
C:\>bitsadmin /Peers /Clear
```
