# bitsadmin peers and clear

Applies to Windows Server (All supported versions)

The `bitsadmin /peers /clear` command clears the peer list.

## Syntax

```
bitsadmin /peers /clear
```

## Examples

To clear the peer list.

```
bitsadmin /peers /clear
```
