# nslookup set all



Prints the current values of the configuration settings.

## Syntax

```
set all 
```

## Parameters

| Parameter | Description |
|-----------|-------------|
|   {help   |     ?}      |

## Remarks

-   **Set all** also prints information about the default server and computer (that is, the host).
