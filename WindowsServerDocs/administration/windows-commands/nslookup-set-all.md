# nslookup set all

Outputs the current configuration setting values, including the default server and computer (the host).

## Syntax

```
set all
```

### Parameters

| Parameter | Description |
| --------- | ----------- |
| /? | Displays help at the command prompt. |
| /help | Displays help at the command prompt. |
