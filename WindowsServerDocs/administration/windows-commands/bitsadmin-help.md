# bitsadmin help

Applies to: Windows Server (All supported versions)

The bitsadmin help command displays help-related information about the bitsadmin command-line parameters and options.

## Syntax

```
bitsadmin /help | /?
```

## Examples

To retrieve the command-line help.

```
bitsadmin /help
```
