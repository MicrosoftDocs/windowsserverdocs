# bitsadmin peercaching and help

Displays the command-line usage for the **/peercaching** switches.

## Syntax

```
bitsadmin /peercaching /help
```

## Examples

To display the command-line help for the **/peercaching** switches:

```
bitsadmin /peercaching /help
```
