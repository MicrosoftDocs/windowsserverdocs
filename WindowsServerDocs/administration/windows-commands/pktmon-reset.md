# pktmon reset

> Applies to: Windows Server (Semi-Annual Channel), Windows Server 2019, Windows 10, Azure Stack HCI, Azure Stack Hub, Azure

Reset counters to zero.

## Syntax

```
pktmon reset [-counters]
```
