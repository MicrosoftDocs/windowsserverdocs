# telnet: display

>Applies to: Windows Server 2022, Windows Server 2019, Windows Server 2016, Windows Server 2012 R2, Windows Server 2012

Displays operating parameters.

## Syntax

```
d[isplay]
```

## Examples

To display operating parameters, type:

```
d
```
