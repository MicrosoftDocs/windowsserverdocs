# telnet: display

>Applies To: Windows Server (Semi-Annual Channel), Windows Server 2016, Windows Server 2012 R2, Windows Server 2012

Displays operating parameters.   
## Syntax  
```  
d[isplay]  
```  
### Parameters  
none  
## <a name="BKMK_Examples"></a>Examples  
Display operating parameters.  
```  
d  
```
