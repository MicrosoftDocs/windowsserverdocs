# cls

Clears the Command Prompt window.

## Syntax

```
cls
```

### Parameters

| Parameter | Description |
| --------- | ----------- |
| /? | Displays help at the command prompt. |

## Examples

To clear all information that appears in the Command Prompt window and return to a blank window, type:

```
cls
```
