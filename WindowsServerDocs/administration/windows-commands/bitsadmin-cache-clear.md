# bitsadmin cache and clear

Applies to: Windows Server (All supported versions)

The bitsadmin cache and clear command purges the local cache.

## Syntax

```
bitsadmin /cache /clear
```

## Examples

To purge the local cache:

```
bitsadmin /cache /clear
```
