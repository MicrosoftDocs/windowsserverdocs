# bitsadmin peers and discover



Discovers peers again.

## Syntax

```
bitsadmin /Peers /Discover 
```

## <a name="BKMK_examples"></a>Examples

The following example discovers peers.
```
C:\>bitsadmin /Peers /Discover
```
