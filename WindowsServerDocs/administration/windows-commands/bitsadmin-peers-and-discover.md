# bitsadmin peers and discover

Applies to: Windows Server (All supported versions)

The bitsadmin peers and discover command rediscovers peers.

## Syntax

```
bitsadmin /peers /discover
```

## Examples

To rediscover peers:

```
bitsadmin /peers /discover
```
