# bitsadmin cache and getexpirationtime

Retrieves the cache expiration time.

## Syntax

```
bitsadmin /cache /getexpirationtime
```

## Examples

To retrieve the cache expiration time:

```
bitsadmin /cache /getexpirationtime
```
