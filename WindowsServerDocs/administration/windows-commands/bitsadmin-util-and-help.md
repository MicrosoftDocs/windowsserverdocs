# bitsadmin util and help



Displays the command-line usage for the **Util** switches.

## Syntax

```
Bitsadmin /Util /Help 
```

## <a name="BKMK_examples"></a>Examples

The following command displays the command-line help for the **Util** switches.
```
Bitsadmin /Util /Help
```
