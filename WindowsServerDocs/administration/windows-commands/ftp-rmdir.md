# ftp rmdir



Deletes a remote directory.

## Syntax

```
rmdir <directory>
```

### Parameters

| Parameter | Description |
| --------- | ----------- |
| `<directory>` | Specifies the name of the remote directory to delete. |

### Examples

To delete the *pictures* remote directory, type:

```
rmdir pictures
```
