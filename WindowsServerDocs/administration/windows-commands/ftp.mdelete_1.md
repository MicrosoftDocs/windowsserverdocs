# ftp mdelete



Deletes files on the remote computer.

## Syntax
```
mdelete <remotefile>[...]
```

### Parameters

| Parameter | Description |
| --------- | ----------- |
| `<remotefile>` | Specifies the remote file to delete. |

### Examples

To delete remote files *a.exe* and *b.exe*, type:

```
mdelete a.exe b.exe
```
