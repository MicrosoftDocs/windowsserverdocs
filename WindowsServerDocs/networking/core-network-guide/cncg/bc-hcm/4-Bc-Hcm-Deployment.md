- [Install the BranchCache Feature and Configure the Hosted Cache Server by Service Connection Point](5-Bc-Feature-Scp.md)

- [Move and Resize the Hosted Cache &#40;Optional&#41;](6-Bc-Move-Resize-Cache.md)
