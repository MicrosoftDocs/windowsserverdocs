# Post-upgrade

1. After Setup has completed installing and rebooted the sever, login to check that the server has been successfully upgraded. If you selected Windows Server 2019 Standard or Datacenter Desktop Edition, you should see the Server Manager window:

    ![](https://raw.githubusercontent.com/coreyp-at-msft/ws-upgrade-center/dev/en-US/media/4d5f364430795e99cdf26e878730e3d5.png)

2. Run **RegEdit** and check the value of the

    **HKEY_LOCAL_MACHINE\\SOFTWARE\\Microsoft\\WindowsNT\\CurrentVersion** hive â€“ the version should be successfully updated to Windows Server 2019.

    ![](https://raw.githubusercontent.com/coreyp-at-msft/ws-upgrade-center/dev/en-US/media/2f2c2c86f36543f8459f736b9aaeb14c.png)

3. Check that applications are running and that client connections to the applications succeed.

4. If you believe that there has been an issue upgrading you server, you should copy and Zip the **%SystemRoot%\\Panther** (usually **C:\\Windows\\Panther**) directory and contact support.
